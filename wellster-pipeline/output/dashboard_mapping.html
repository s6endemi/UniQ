The Problem We Solved

Every time Wellster updates a survey, questions get new IDs — even when they ask the exact same thing. We reduced 677 fragmented IDs into 15 clean, analytics-ready clinical categories.

ID Fragmentation by Category

How many different IDs exist per actual unique question? Higher = more survey versions existed. COMORBIDITY_SCREENING is the worst: 10 question texts spread across 116 different IDs.

Data Coverage per Category

How many rows and patients each category covers. COMORBIDITY_SCREENING dominates because every patient gets screened for pre-existing conditions.

Answer Formats

Each answer type needs a different parser in Step 5. Predefined = multiple choice, JSON = structured BMI data, File = photo uploads, Confirmation = consent acknowledgment.
